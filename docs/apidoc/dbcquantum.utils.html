dbcquantum.utils.bin_frac_to_dec(binary_fraction: list[int] | list[bool], reverse: bool = False) → float[source]¶

Convert a binary fraction to a decimal number.

Parameters:

  • binary_fraction (Union[List[int], List[bool]]) – The binary fraction to be converted.

  • reverse (bool, optional) – If True, reverse the order of the binary fraction before conversion. Defaults to False.

Returns:

The decimal representation of the binary fraction.

Return type:

float

Examples

>>> print(binary_fraction_to_decimal([1,0,1,1], reverse = False))
0.8125

>>> print(binary_fraction_to_decimal([1,0,1,1], reverse = True))
0.6875

dbcquantum.utils.partial_state(state: Statevector, focus_qubits: Iterable[int]) → Statevector[source]¶

Extract the state of some qubits from the entire quantum state. You can specify the qubits whose state you want to know.

Parameters:

  • state (Statevector) – The entire quantum state.

  • focus_qubits (Iterable[QubitSpecifier]) – Qubits whose state is to be extracted. The order is reflected in the returned statevector.

Returns:

The state of the specified qubits.

Return type:

Statevector

Examples

>>> print(eq_state(One ^ Minus,
                   partial_state(
                        to_Statevector(Minus ^ Plus ^ One ^ Zero),
                        [3, 1])))
True
